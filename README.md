# Linux System Administration Lab

## Project Overview
This project demonstrates fundamental Linux system administration skills using Ubuntu 24.04 LTS in a VirtualBox virtual machine. It covers user and group management, file permissions, system monitoring, networking, and Bash scripting.

## Objectives
- Practice Linux command-line administration.
- Manage users, groups, and file permissions.
- Monitor system resources.
- Configure and verify network connectivity.
- Create a simple Bash automation script.

## Environment
- Ubuntu 24.04.4 LTS
- Oracle VirtualBox

## Tasks Performed

### User and Group Management
- Created a new user (`sarah`).
- Created the `hr` group.
- Added the user to the HR group.
- Verified user and group membership.

### File and Permission Management
- Created `employees.txt`.
- Changed group ownership.
- Modified file permissions using `chmod`.
- Managed ownership using `chown`.
- Organized files under `/srv/techsolutions/hr`.

### System Monitoring
- Checked disk usage using `df -h`.
- Monitored memory usage using `free -h`.
- Displayed system information with `hostnamectl`.
- Verified operating system details.

### Networking
- Displayed network interfaces using `ip a`.
- Verified routing table using `ip route`.
- Tested internet connectivity using `ping`.

### File Search
- Located files using `find`.
- Located executable files using `which`.
- Retrieved command locations using `whereis`.

### Bash Automation
Created a Bash script (`system_report.sh`) that displays:
- Hostname
- Operating System
- Disk Usage
- Memory Usage
- IP Address
